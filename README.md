# Kiosk-by-C
## :mag: 주제
:sparkles: C언어로 카페 키오스크 개발 (SW융합코딩 수업 팀프로젝트) :sparkles:

## :pushpin: 목적
학교 근처에 위치한 "쥬씨" 카페의 키오스크을 대상으로 주요 기능들을 c언어로 개발.
